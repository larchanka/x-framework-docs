Creating themes

X-Framework uses the LESS preprocessor to simplify development. The styles in X-Framework are separated into files so that unnecessary styles could be easily removed. The master file is xf.less, where all other files are imported to. It's the only file that needs to be compiled for production use. We recommend importing your stylesheet in this file too. Include at the end of all imports, but before xf.utilities.less.

To customize look and feel of an application you need to edit variables in xf.variables.less. Variables have descriptive names and are grouped logically.
