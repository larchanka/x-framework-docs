Tabs Component

To insert tabs use the following snippet:

<div data-component="TabsComponent"
    data-id="app-tabs">
    <script>
        XF.setOptionsByID('app-tabs',
             { tabsPerRow: 4,
               tabsData: [{
                  text:'Four',
                  active: true
                }, {
                  text:'beautiful'
                }, {
                  text:'tabs'
                }, {
                  text:'!'
                }]
             }
        );
    </script>
</div>

Using tabsPerRow parameter you can define how many tabs should be displayed per row. Tabs component is smart. You've no need to worry about positioning tabs elements.

The component has the following configurable properties.

tabsPerRow: 4
Maximum 6. How many tabs to fit int one row.
tabsClass: ''
A custom class name to be set on every tab.
tabsData: []
An array of objects representing individual tabs. Each one can have the following properties:
text: ''
Text to be displayed on the button
active: false
Indicates whether the tab is active (selected).
