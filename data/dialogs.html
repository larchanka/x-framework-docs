Dialogs

There are two kinds of dialogs a in X-Framework: interactive and simple loading notifications.

To show a dialog call: 

XF.UIElements.showDialog(headerText, messageText, buttons);

To hide it call: 

XF.UIElements.hideDialog();
headerText String
Text that will be shown as the dialog title
messageText String
Text that will be shown as the dialog message text
buttons Array
An array of buttons to show (jQuery objects or objects with button description for createButton() method. Each button in the buttons array can have the following properties:
text String
Button text
icon String
Button icon name
iconpos String
Button icon position. Possible values: "top", "bottom", "right", "left"
small Boolean
Whether this is a small button
special Boolean
Whether the button should be of an accented color
alert Boolean
Whether the button should be styled as vey important, red by default
handler Function
Function that will be called when the button is pressed

The buttons argument may be omitted, in this case there will be an OK button which will close the dialog.

To show a loading dialog call

XF.UIElements.showLoading(messageText, iconName)
messageText String
Text that will be shown as the dialog message text. default is "Loading..."
icon String
Button icon name

To hide it call: 

XF.UIElements.hideLoading()
