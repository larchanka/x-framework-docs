Adding icons

X-Framework provides a number of vector icons, which can be added to buttons and list elements. To add an icon just set [data-icon=iconname] attribute to the necessary element and icon wil be rendered.

To change icon position set [data-iconpos=left|right|top|bottom] attribute to the same element. By default the icon will be on the left side.

<a data-appearance="button"
   data-icon="tick"
   data-iconpos="right"
   href="#">
	[data-iconpos=right][data-icon=tick]
</a>
[data-iconpos=right] [data-icon=tick]
